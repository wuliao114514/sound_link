# Sound Link 使用教程

## 快速开始

### 首次启动

1. **启动应用**：双击 Sound Link 应用图标
2. **自动安装依赖**：首次启动会自动检测并安装 `AudioDeviceCmdlets` PowerShell 模块
3. **等待初始化**：首次启动可能需要几秒钟时间来检测系统音频设备

### 主界面介绍

启动后，你会看到 Sound Link 的主界面，包含：

- **中心球**：显示当前模式（管理/路由）
- **设备球**：环绕显示系统中的音频设备
- **顶部按钮**：模式切换和设置按钮

## 基本操作

### 管理模式

管理模式是默认模式，用于快速切换音频输出设备。

#### 切换音频设备

1. **点击设备球**：点击任意设备球即可将其设置为默认音频输出设备
2. **查看当前设备**：被激活的设备球会显示为高亮状态
3. **切换回默认设备**：点击中心球可切换回系统默认设备

#### 设备管理

1. **右键点击设备球**：可以打开设备设置面板
2. **调整设备音量**：在设置面板中可以调整单个设备的音量
3. **设置延迟**：为设备添加延迟，适用于音频同步场景

### 路由模式

路由模式允许将系统音频同时输出到多个设备。

#### 启用路由模式

1. **点击顶部路由按钮**：切换到路由模式
2. **安装 VB-Cable**：如果未安装 VB-Cable，会自动弹出安装提示
3. **选择目标设备**：点击设备球将其添加到广播目标列表

#### 开始广播

1. **点击中心球**：开始音频广播
2. **查看广播状态**：中心球会变为绿色并显示脉冲动画
3. **管理目标设备**：
   - 再次点击设备球可移除广播目标
   - 可以同时选择多个设备进行广播

#### 停止广播

1. **点击中心球**：停止音频广播
2. **自动恢复**：系统会自动恢复到原默认音频设备

## 高级功能

### 系统托盘

Sound Link 会在系统托盘中常驻：

- **左键点击**：显示主界面
- **右键点击**：打开快捷菜单
  - **显示**：显示主界面
  - **退出**：退出应用

### 主题设置

1. **打开设置**：点击顶部设置按钮
2. **主题选项**：
   - **跟随系统**：自动使用系统主题（推荐）
   - **深色模式**：强制使用深色主题
   - **浅色模式**：强制使用浅色主题
3. **高级材质**：启用毛玻璃效果（仅在支持的系统上可用）

### 音频路由高级设置

1. **设备设置**：为每个目标设备单独设置音量和延迟
2. **延迟调整**：用于解决不同设备间的音频同步问题
3. **音量同步**：可以选择是否同步所有设备的音量

## 常见问题

### VB-Cable 安装问题

**问题**：无法检测到 VB-Cable

**解决方案**：
1. 确保已正确安装 VB-Cable
2. 重启 Sound Link 应用
3. 如有必要，重启计算机后再试

### 设备检测问题

**问题**：某些音频设备未显示

**解决方案**：
1. 点击设置中的 "刷新设备列表"
2. 检查设备是否已正确连接到系统
3. 确保设备驱动已正确安装

### 权限问题

**问题**：无法切换音频设备

**解决方案**：
1. 以管理员身份运行 Sound Link
2. 检查 PowerShell 执行策略设置

### 音频路由问题

**问题**：路由启动失败

**解决方案**：
1. 确保 VB-Cable 已正确安装
2. 检查是否有其他应用正在占用 VB-Cable
3. 尝试重启 Sound Link 应用

## 最佳实践

### 多设备场景

**家庭影院**：
- 将电视和音响同时添加为广播目标
- 为不同设备设置适当的延迟以实现同步

**游戏直播**：
- 将耳机和直播麦克风同时添加为目标
- 调整音量以获得最佳直播效果

### 性能优化

- **减少目标设备数量**：同时广播到过多设备可能会影响性能
- **合理设置延迟**：过高的延迟可能导致音频不同步
- **关闭未使用的设备**：在管理模式下禁用不常用的设备

## 故障排除

### 应用崩溃

1. **检查日志**：查看应用日志文件（位于 `%APPDATA%\sound-link\logs`）
2. **重新安装**：卸载并重新安装应用
3. **检查系统兼容性**：确保系统满足最低要求（Windows 10/11）

### 音频质量问题

1. **检查设备驱动**：确保所有音频设备驱动都是最新版本
2. **调整采样率**：在系统音频设置中调整采样率
3. **检查 VB-Cable 设置**：确保 VB-Cable 配置正确

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Alt + X` | 退出应用 |
| `Alt + S` | 显示/隐藏设置面板 |
| `Alt + M` | 切换模式（管理/路由） |
| `Alt + R` | 开始/停止广播（路由模式） |
